## Requisitos

- Node.js v18.20.8
- [Expo](https://expo.dev/) (usando `npx expo`)

## Instalación

1. Clona el repositorio:
  ```bash
  git clone https://github.com/rauljunior777/rjr-player.git
  ```
2. Instala las dependencias:
  ```bash
  cd rjr-player
  npm install
  ```

## Uso

Para iniciar el proyecto, ejecuta:
```bash
npx expo start
```
